# Landsfy - Real Estate Marketplace

Landsfy is a premium, high-performance real estate marketplace designed to connect property buyers, sellers, and agents seamlessly. Built with a focus on speed, responsiveness, and a premium user experience.

## 🚀 Features

- **Dynamic Property Search:** Advanced filtering by location, price, purpose, and property type.
- **Premium UI/UX:** Mobile-first, responsive design with smooth animations and interactive elements.
- **Interactive Gallery:** High-quality image sliders and thumbnail navigation.
- **Agent & Agency Portals:** Dedicated dashboards for managing listings and leads.
- **Google OAuth Integration:** Secure and easy login for users.
- **Similar Properties:** Intelligent recommendations based on location and category.
- **Detailed Analytics:** Track views and interactions for every listing.

## 🛠️ Tech Stack

- **Frontend:** HTML5, Vanilla CSS3, JavaScript (ES6+), FontAwesome.
- **Backend:** PHP (PDO for secure database interactions).
- **Database:** MySQL.
- **Integrations:** Google OAuth API, GSAP (Animations).

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/akamaanullah/landsfy.git
   ```
2. Configure your environment:
   - Rename `.env.example` to `.env`.
   - Update your database credentials and Google API keys.
3. Import the database:
   - Import the provided SQL file (if available) into your MySQL server.
4. Run locally:
   - Ensure you are using XAMPP/WAMP or a similar PHP environment.
